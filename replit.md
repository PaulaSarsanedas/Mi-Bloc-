# Mi-Bloc - Laboratorio en la Nube

## Overview
This is a cloud-based laboratory project ("Mi-Bloc") set up as a simple static website starter. The project was imported from GitHub and configured to run in the Replit environment.

## Project Structure
- `index.html` - Main HTML page with Spanish content
- `style.css` - Styling with gradient background and card layout
- `script.js` - Basic JavaScript for animations
- `server.py` - Python HTTP server configured for port 5000 with cache control

## Current State
- Simple static website with welcome page
- Responsive design with feature cards
- Python HTTP server for serving static files
- Configured for Replit environment on port 5000

## Technical Setup
- **Language**: Python 3.11
- **Server**: Python's built-in HTTP server
- **Port**: 5000 (frontend)
- **Host**: 0.0.0.0 (required for Replit proxy)
